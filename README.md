## 🚀 Getting Started

1. **Clone this repository**
   ```bash
   git clone https://github.com/jay240322/python.git
   ```

2. **Navigate to the folder**
   ```bash
   cd python
   ```

3. **Run any Python file**
   ```bash
   python filename.py
   ```

> ✅ Make sure you have **Python 3.x** installed on your system.
